# Bihar AI Summit 2026 - Event Analytics Dashboard

## Overview

This project is an interactive Power BI dashboard developed during my Data Analytics Internship to analyze participant registrations and attendance for Bihar AI Summit 2026.

The dashboard transforms raw registration data into meaningful insights through KPI cards, trend analysis, participant segmentation, and geographical distribution visualizations.

---

## Dashboard Highlights

### Key Performance Indicators (KPIs)

- Total Profiles: 5,587
- At Event: 3,923
- Student Participants: 4,491
- Professional Participants: 610
- Startup Participants: 445
- Government Participants: 39
- Blocked Profiles: 43
- Attendance Rate: 70.22%
- Total Cities: 239
- Total Organizations: 1,408

---

## Features

### Registration Trend Analysis
Tracks registration growth over the registration period and highlights peak registration days.

### Top Contributing Cities
Displays the cities generating the highest number of registrations.

### Guest Type Allocation
Visualizes participant distribution among:
- Students
- Professionals
- Startups
- Government Representatives
- Sponsors

### Attendance Analytics
Compares total registrations with actual event attendance and calculates attendance percentage.

### Interactive Dashboard
Provides an easy-to-understand overview of event participation through dynamic visualizations and KPI cards.

---

## Dataset

The dashboard uses participant registration data containing:

- Name
- Email
- Mobile Number
- City
- State
- Organization
- Designation
- Guest Type
- QR Serial Number

Data cleaning and transformation were performed using Power Query and DAX.

## Key Insights

- Students represented the largest participant category.
- Patna was the leading contributor to registrations.
- Attendance exceeded 70% of total registrations.
- Registration activity increased significantly closer to the event date.
